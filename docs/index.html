🎉 חדש בגרסה 2.0.0

  • התרעות פיקוד העורף בזמן אמת — עם צליל, התראת מערכת, היסטוריה וסינון לפי עיר
  • התראות זמני יום — לחיצה ימנית על כל זמן להגדרת תזכורת X דקות לפני
  • 5 צלילי התראה לבחירה — צלצול, פעמון, עדין, דחוף, שקט + תצוגה מקדימה
  • אירועים מ-iCal / Google Calendar — סנכרון אוטומטי עם כתובת iCal
  • ניטור מערכת — צריכת CPU ו-RAM בזמן אמת
  • ספירת העומר — בחירת אותיות (גימטריה) או מספרים + נוסח לעומר/בעומר
  • ערכות נושא — Dracula, Nord, Solarized
  • חסימת מקורות חדשות ומילות מפתח
  • גרירת כפתור הפעלה למיקום חופשי על המסך
  • קיצור מקשים גלובלי לפתיחת הפאנל
  • מניעת הפעלה כפולה של התוכנה
  • תיקון באג קריטי — ספריית ווידג'טים עובדת כמו שצריך
  • הפעלה עם Windows — תמיכה מלאה גם בגרסת פיתוח
